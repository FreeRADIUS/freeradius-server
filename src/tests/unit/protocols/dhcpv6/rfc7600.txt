#  Figure 9: Encapsulated Option for Non-Mapping-Rule Parameters of 4rd Domains
#
#  o  option code: 99, encapsulated OPTION_IP4RD_NON_MAP_RULE option (see Section 6)
#  o  option-length: 4
#  o  H: Hub-and-spoke topology (= 1 if Yes)
