▸i How the median (p50) baseline is chosen
  1. Sum CEst across all functions for each of the 0 selected runs → one total CEst per run.
  2. Rank the runs by total CEst and take the median (p50) run — the middle one (for an even count, the lower-middle). That run becomes the baseline column.
  3. Every cell's Δ is measured against that baseline run, per function:
Δ%(fn, run) = ( CEstfn,run − CEstfn,p50 ) / CEstfn,p50 × 100 where p50 = argmin over runs of | rank(total CEst) − N/2 |

Why a real run and not the mean: the p50 run is an actual profile (robust to outliers), so every comparison is run-vs-run rather than run-vs-synthetic-average. The baseline is recomputed whenever the selected set changes.
