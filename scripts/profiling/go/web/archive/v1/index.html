CPU Cycle Utilization (CEst) Utility

Callgrind CEst analysis in the browser. Everything runs locally via WebAssembly; no file leaves your machine.

Cycle Estimation (CEst) formula

As implemented in QCachegrind/KCachegrind.

CEst = Ir + 10*(I1mr+D1mr+D1mw) + 100*(ILmr+DLmr+DLmw) + 10*(Bcm+Bim)

Profiling runs

Each run is one directory's callgrind.out.* files. Add a second run to get a comparison footer (first run is the baseline).
