= %rpad(<string>, <val>, <char>)

Right-pad a string.

.Return: _string_

.Example

[source,unlang]
----
&control.Tmp-String-0 := "123"

&reply.Reply-Message := "Maximum should be %rpad(%{control.Tmp-String-0}, 11, '0')"
----

.Output

```
Maximum should be 12300000000
```
