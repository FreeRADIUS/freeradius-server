= %str.upper( ... )

Returns the uppercase version of the input.

The uppercase operation is done using the current locale.

.Return: _string_

.Example

[source,unlang]
----
User-Name := "bob"
reply.Reply-Message := "uppercase of %{User-Name} is " + %str.upper(User-Name)
----

.Output

```
uppercase of bob is BOB
```
