= %str.lpad(<string>, <val>, <char>)

Left-pad a string.

.Return: _string_

.Example

[source,unlang]
----
string test

test := "123"

reply.Reply-Message := "Maximum should be %str.lpad(test, 11, '0')"
----

.Output

```
Maximum should be 00000000123
```
